A small but functional npm package
Install
Install using npm install npm-test-jovanna

Usage
In a React app, use the Button or Badge components:
import { Button } from 'npm-test-jovanna'
import { Badge } from 'npm-test-jovanna'

Button props
Name	Description
kind	'Primary'
data-id	string
type	'button'
name	string
value	string
disabled	Boolean
onClick	function
Badge props
Name	Description
value	integer
